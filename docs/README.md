# Docs

Project documentation that isn't code. Drop notes, briefs, decisions, references, and meeting outcomes here.

## Suggested files (create as needed)

- `brief.md` — what the project is, who it's for, success criteria
- `strategy.md` — positioning, messaging, naming
- `decisions/` — short decision records (one file per non-trivial decision)
- `references/` — links, inspiration, competitive analysis
